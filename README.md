# BookMind AI — Intelligent Book Recommendation System

> Full-stack AI platform for semantic book recommendations powered by TF-IDF, Cosine Similarity & Collaborative Filtering.

## Quick Start

### Step 1 — Run the Notebook

```bash
jupyter lab
```

Open `notebook/book_recommendation.ipynb` → Run All.
This generates all models in `models/` (~5 min on first run).

### Step 2 — Start the Backend

```bash
cd backend
bash start.sh
# → http://localhost:8001
```

Or directly:
```bash
cd backend
uvicorn main:app --reload --port 8001
```

### Step 3 — Start the Frontend

```bash
cd frontend
npm install
npm run dev
# → http://localhost:3000
```

## API Reference

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/health` | Health check |
| GET | `/api/books?q=&limit=20` | Search books |
| GET | `/api/books/{isbn}` | Book detail + recommendations |
| POST | `/api/recommend` | Get recommendations by ISBN/title |
| POST | `/api/profile/recommend` | Profile-based recommendations |
| GET | `/api/analytics` | Full analytics data |
| POST | `/api/chat` | AI Chat Librarian (Claude) |

### Example

```bash
# Search books
curl "http://localhost:8001/api/books?q=harry+potter&limit=5"

# Get hybrid recommendations
curl -X POST http://localhost:8001/api/recommend \
  -H "Content-Type: application/json" \
  -d '{"isbn":"059035342X","method":"hybrid","n":10}'

# Analytics
curl http://localhost:8001/api/analytics
```

## ML Models

| Model | Algorithm | Description |
|-------|-----------|-------------|
| Popularity | Bayesian Weighted Average | `(v/(v+m)) × R + (m/(v+m)) × C` |
| Collaborative Filtering | Cosine Similarity (item-item) | User×Book matrix → book similarity |
| Content-Based | TF-IDF + Cosine | Title + Author vectorization |
| Hybrid | Weighted combination | `0.6 × CF + 0.4 × CB` |

### Filtering thresholds
- Books: ≥ 50 ratings → 531 books in model
- Users: ≥ 50 ratings
